CSS Cheatsheet: selectors

Use this cheatsheet to understand CSS selectors. For more in depth and interactive exercises visit W3Schools CSS selectors

Attribute name Tag Description
Element selector element { }

p { font-family: serif; }

where p is the paragraph selector. The CSS applied here will affect all paragraphs unless they are individually overwritten using class selector or id selector.
Class selector .classname { } The class selector selects HTML elements with a specific class attribute. To select elements with a specific class, write a period (.) character, followed by the class name.
Id selector #id { } To select an element with a specific id, write a hash (#) character, followed by the id of the element.
Group selector element1, element2, element3{} To group selectors, separate each selector with a comma.
Universal selector * { } The universal selector (*) selects all HTML elements on the page.
